KazNLP: Қазақ тіліне арналған NLP құралдары

Этот проект направлен на создание бесплатных инструментов обработки языка с открытым исходным кодом для Kazakah . Предлагаемый набор инструментов предназначен для решения широкого круга проблем НЛП, которые можно разделить на процедуры предварительной обработки, базовой обработки и прикладного уровня. Ожидается, что инструменты будут реализованы на языке программирования Python 3 и выпущены под лицензией CC-SA-BY или совместимыми лицензиями.
